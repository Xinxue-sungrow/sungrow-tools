# Sungrow Field Engineering Tools

This repository contains internal tools designed to support field engineers during onsite inspection, troubleshooting, and reporting.

The goal of this repository is to simplify daily engineering tasks, improve reporting efficiency, and standardize field documentation.

These tools are lightweight, easy to use, and require minimal setup so that engineers can quickly access them during field operations.

---

## Current Tools

### Field Action Report Tool

A simple browser-based tool that allows engineers to quickly generate a field report with photos.

Features:

- Input site name
- Select action type (AC Test, DC Test, Inverter Check, etc.)
- Upload inspection photos
- Add multiple actions
- Automatically generate a structured PDF report

The tool runs entirely in the browser and does not require installation.

---

## How to Use

1. Download the tool file.

2. Open the file in any modern browser.
