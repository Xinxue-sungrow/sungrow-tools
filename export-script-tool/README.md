# Parameter Export Script Tool

This tool allows field engineers to quickly export parameter tables from the UMCG interface into a CSV (Excel-compatible) file.

It is designed for fast onsite usage without requiring installation or backend support.

---

## 🔧 Features

- Export visible parameter table (Run-param / Pro-param / Sys-param)
- Supports:
  - Input fields
  - Dropdown selections
  - Plain text values
- Automatically detects active parameter tab
- One-click CSV download
- No installation required

---

## 🚀 How to Use

---
### Step 1
Log in to **UMCG**

---
### Step 2
Navigate to:
Monitoring → Param.setting → Run-param.setting

---
### Step 3
Open browser tools:
More Tools → Developer Tools


---

### Step 4
Go to the **Console** tab

---

### Step 5
Copy and paste the script from:
export-script.js


---

### Step 6
Press:
Enter


---

### Step 7
The CSV file will be automatically downloaded
